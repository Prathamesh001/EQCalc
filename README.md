# 🏢 IS 1893 Seismic Base Shear Calculator

![AI Powered](https://img.shields.io/badge/AI_Powered-Llama_4_Scout-046A38?style=for-the-badge)
![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)

An intelligent structural engineering tool that bridges traditional finite element mechanics with modern Vision AI. Automate your seismic base shear calculations (ESM & RSM) instantly from architectural drawings.

### 🚀 Try the Live App: [eqcalc.streamlit.app](https://eqcalc.streamlit.app/)

## ✨ What it does
* **Omni-Reader AI:** Upload plans, elevations, or notes. The Llama 4 Scout AI extracts geometry, seismic parameters (Z, I, R), and structural matrices automatically.
* **IS 1893 Compliance:** Calculates Equivalent Static Method (ESM) and Response Spectrum Method (RSM) base shears, applying code-mandated scaling factors.
* **Dynamic Visualization:** Instantly plots the first 3 mode shapes and mass participation data.

## 🛠️ How to Use
1. Open the [Web App](https://eqcalc.streamlit.app/).
2. Upload your structural drawings (JPG, PNG, or PDF).
3. Hit **Extract Data** and run your analysis!
